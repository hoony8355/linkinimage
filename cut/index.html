이미지 분할 링크 생성기

하나의 이미지를 원하는 위치에서 나누고, 각 조각마다 다른 링크를 삽입할 수 있는 도구입니다. 생성된 HTML 코드는 뉴스레터, 이메일, 블로그 등에 바로 붙여넣을 수 있습니다.

🧭 사용 방법 안내

  1. 이미지 업로드: 분할할 이미지를 불러옵니다.
  2. 분할선 추가: 이미지 위를 클릭하면 가로선이 생깁니다. 이 선을 기준으로 이미지가 나뉩니다.
  3. 링크 입력 & 코드 생성: 각 조각에 링크를 입력한 후 HTML 코드를 생성합니다.
  4. HTML 복사 & 활용: 생성된 코드를 복사해 원하는 곳에 붙여넣습니다.


이미지 위를 클릭하여 원하는 위치에 분할선을 추가해보세요.
