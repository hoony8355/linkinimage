링킨이미지 (Linkin Image)

🔍 사용 방법 안내

  1. 이미지 업로드 또는 URL 입력을 통해 이미지를 불러옵니다.
  2. + 버튼 추가를 눌러 링크를 넣을 영역을 생성합니다.
  3. 생성된 박스를 원하는 위치와 크기로 조절합니다.
  4. 원하는 링크 주소와 제목을 입력합니다.
  5. 코드 생성 버튼을 눌러 CSS 또는 HTML map 코드를 복사합니다.

🧩 버튼 설명

  • ✏️ 수정 – 현재 영역에 설정된 링크 주소와 제목을 변경합니다.
  • ❌ 삭제 – 해당 링크 영역을 삭제합니다.
  • 📐 크기조절 – 클릭 후 드래그하여 박스의 크기를 변경할 수 있는 모드를 토글합니다.
  • 🔍 확대/축소 – 전체 이미지를 축소하거나 원래 크기로 되돌릴 수 있습니다.
  • 🔧 슬라이더 – 이미지 및 박스 전체를 비율에 맞게 확대/축소 조정할 수 있습니다.
